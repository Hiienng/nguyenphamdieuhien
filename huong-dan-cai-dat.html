Phần 1

Cài Đặt Extension

Extension chạy trên Firefox và hoạt động âm thầm trong nền — bạn không cần thao tác thêm sau khi cài xong. Mỗi lần bạn mở trang Etsy Ads, dữ liệu sẽ tự động được ghi lại.

▶
Video hướng dẫn — Cài đặt Extension https://drive.google.com/file/d/1S7rx1EegJ482Ra7Jy5_MzMmurvFVdWF_/view?usp=sharing
1
Tải file Extension
Nhấp vào đường dẫn bên dưới để tải file .xpi về máy. Trình duyệt sẽ hỏi bạn muốn lưu file ở đâu — hãy chọn bất kỳ vị trí nào dễ tìm.
Link tải Extension https://github.com/Hiienng/nguyenphamdieuhien/releases/download/extension/getifyco-extension.xpi
2
Mở trang quản lý Add-on trong Firefox
Trên Firefox, nhập địa chỉ sau vào thanh URL rồi nhấn Enter:
about:addons
3
Cài từ file
Trong trang Add-ons, nhấp vào biểu tượng bánh răng ⚙ (góc trên bên phải), chọn "Install Add-on From File…", sau đó chọn file .xpi bạn vừa tải. Firefox sẽ hỏi xác nhận — nhấn Add.
4
Cấu hình kết nối
Sau khi cài, biểu tượng Extension xuất hiện trên thanh công cụ Firefox. Nhấp vào để mở popup, chọn Settings, điền thông tin kết nối được cung cấp bởi GetifyCo, rồi nhấn Save.
5
Kiểm tra hoạt động
Mở trang etsy.com và truy cập mục Ads Manager. Quay lại popup Extension — nếu thấy dòng "Ready" và số lượng responses tăng lên, Extension đã hoạt động đúng.
✓
Extension hoạt động âm thầm — bạn không cần mở popup mỗi lần dùng. Chỉ cần để Firefox mở và duyệt Etsy như bình thường.
Phần 2

Cài Đặt Ứng Dụng

GetifyCo Listing Portal là ứng dụng desktop chạy trực tiếp trên máy tính Windows. Không cần cài thêm Python hay bất kỳ phần mềm nào khác — chỉ cần giải nén và chạy.

▶
Video hướng dẫn — Cài đặt Ứng dụng https://drive.google.com/file/d/1LfscSOHxzPDVIb5hffoyjCKBuk57uLFf/view?usp=sharing
1
Tải file cài đặt
Nhấp vào đường dẫn bên dưới để tải file .zip về máy.
Link tải App (Windows) https://github.com/Hiienng/nguyenphamdieuhien/releases/latest/download/getifyco-listing-portal-windows.zip
2
Giải nén toàn bộ file
Nhấp chuột phải vào file .zip vừa tải, chọn "Extract All…" và chọn thư mục lưu trữ (ví dụ: C:\GetifyCo\).
⚠
Không chạy file trực tiếp từ bên trong file .zip. Bạn phải giải nén ra trước — nếu chạy từ trong .zip, ứng dụng sẽ báo lỗi thiếu thư mục _internal.
3
Chạy ứng dụng
Vào thư mục vừa giải nén, tìm file getifyco-listing-portal.exe và nhấp đúp để chạy.
4
Xử lý cảnh báo Windows SmartScreen
Windows có thể hiển thị thông báo "Windows protected your PC". Đây là cảnh báo thông thường với phần mềm chưa được ký số. Nhấn "More info" → "Run anyway" để tiếp tục.
ℹ
Cảnh báo này chỉ xuất hiện lần đầu. Từ lần thứ hai, bạn chỉ cần nhấp đúp vào file .exe là ứng dụng khởi động ngay.
5
Nhập thông tin kết nối cơ sở dữ liệu
Khi ứng dụng khởi động lần đầu, trình duyệt sẽ tự mở trang app. Nhấp vào biểu tượng ⚙ Settings, điền chuỗi kết nối được cung cấp bởi GetifyCo vào ô Database connection (Neon), rồi nhấn Lưu.
✓
Thông tin kết nối chỉ cần nhập một lần duy nhất — ứng dụng lưu lại trên máy bạn và tự động dùng lại từ lần mở sau.
6
Sử dụng ứng dụng
Sau khi kết nối thành công, trang dashboard sẽ tải dữ liệu listing và hiển thị báo cáo hiệu suất. Cửa sổ console nhỏ trên thanh taskbar là máy chủ nội bộ của ứng dụng — hãy để thu nhỏ, không cần đóng. Khi muốn thoát hoàn toàn, đóng cửa sổ console đó.
