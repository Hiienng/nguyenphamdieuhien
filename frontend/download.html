Tải GetifyCo Listing Portal

App desktop chạy server ngay trên máy bạn — không cần trình duyệt hay domain. Mở app, dán access token là dùng được. (Cần kết nối internet để truy cập dữ liệu.)

Cài đặt & mở app

  1. macOS: giải nén → kéo getifyco-listing-portal.app vào Applications. Lần đầu mở: chuột phải → Open (vì app chưa ký số).
  2. Windows: giải nén → chạy getifyco-listing-portal.exe. Nếu SmartScreen cảnh báo: More info → Run anyway.
  3. App mở cửa sổ đăng nhập → dán access token (SECRET_KEY) → vào Listing Manager.

Bản tải đã cấu hình sẵn database — mở là dùng (đổi sau trong Settings của app).
Việc tải yêu cầu đăng nhập GitHub có quyền truy cập (liên hệ admin để được cấp quyền). Xem tất cả phiên bản tại trang Releases.
