Import Report

1

Download & install the extension

Download Extension (.xpi)

Compatible with Firefox. After downloading, rename the file from .zip → .xpi, then in Firefox open about:debugging → This Firefox → Load Temporary Add-on → select the .xpi file.

2

Dán Database connection

Mở popup extension → Database Settings → dán Database connection (cùng chuỗi Neon mà app đang dùng) → Save → Test Connection.

Extension ghi dữ liệu thẳng vào Neon của bạn — không cần API/token. Chuỗi kết nối lưu locally trên máy.

3

Quét Etsy Ads & đồng bộ

Mở Etsy Ads dashboard — extension tự bắt dữ liệu. Bấm Add to DB để đẩy số liệu listing, hoặc Add Keywords to DB cho keyword. Dữ liệu vào thẳng Neon của bạn.

⚡ Fast & free — no credits needed. Captures data while you browse normally. Note: browser automation may be detected by Etsy in future policy updates.
